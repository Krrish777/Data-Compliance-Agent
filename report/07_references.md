## F. Note on Reproducibility of Citations

Stable identifiers (arXiv IDs, DOIs, ISBNs, official `eur-lex` and `owasp.org` URLs) have been preferred wherever available, since vendor documentation roots are subject to periodic re-organisation. Where a citation points to an evolving documentation site (e.g. Section D, references [32]–[46]), the URL refers to the canonical landing page; readers requiring a frozen version should consult the Internet Archive Wayback Machine (https://web.archive.org/) using the access date of **17 April 2026**.
